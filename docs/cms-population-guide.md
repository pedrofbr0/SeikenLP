# Guia para Popular o CMS

## 1. Perfil

Cadastre primeiro o Single Type `Profile` com:

- Nome: Lucas Rodrigues
- Título: Desenhista Cadista | Projetista CAD/BIM | Modelagem 3D e Renderização
- Resumo profissional revisado
- Telefone, e-mail, cidade/UF, LinkedIn e portfólio quando forem oficiais
- `heroImage`: render principal em alta qualidade, preferencialmente 1920x1080 ou maior

## 2. Projetos

Para cada projeto, publique pelo menos:

- `title`
- `slug`
- `category`
- `shortDescription`
- `fullDescription`
- `coverImage`
- `softwareUsed`
- `servicesProvided`
- `highlights`

Priorize imagens horizontais para o `coverImage`, com boa iluminação, nitidez e composição que mostre materiais, profundidade e escala. Galerias podem misturar render final, variações de vista, detalhes de materiais, pranchas e capturas do modelo 3D.

## 3. Categorias recomendadas

- Interiores
- Arquitetônico
- Industrial
- Paisagismo
- Reforma e restauro
- Renderização
- CAD/BIM

## 4. Texto de projeto sugerido

Use uma estrutura curta e comercial:

```txt
Objetivo do projeto:
Descrever a necessidade do ambiente, cliente ou equipe.

Solução técnica:
Explicar desenho técnico, levantamento, modelagem, renderização ou documentação.

Resultado visual:
Destacar materiais, luz, apresentação e clareza para tomada de decisão.
```

## 5. SEO por projeto

Use `seoTitle` com até 60 caracteres e `seoDescription` com até 155 caracteres. Inclua termos naturais como desenho técnico arquitetônico, modelagem 3D, renderização arquitetônica, SketchUp, Lumion, AutoCAD, Revit, CAD/BIM e interiores quando fizer sentido para o projeto.
